'Open Sans, sans-serif'

foodpanda
300

foodpanda

Font Sizes

Use these classes to change the size of our font. It is recommended to stick with font-m for paragraphs, font-l for text that needs emphasis but is not a heading, font-xl and up for headings and font-s and smaller for legal text or text we don't want to put any emphasis on.

@font-size-xs

This is extra small
The font size is 10px

@font-size-s

This is small
The font size is 12px

@font-m

This is medium
The font size is 14px

@font-size-l

This is large
The font size is 18px

@font-size-xl

This is extra large
The font size is 24px

@font-size-xxl

This is extra extra large
The font size is 30px

@font-size-xxxl

This is extra extra extra large
The font size is 36px

Line height

Use these classes to change the line

@line-height-none

Line height is 0 em.

@line-height-xs

Line height is 1.1 em.

@line-height-s

Line height is 1.2 em.

@line-height-m

Line height is 1.3 em.

@line-height-l

Line height is 1.4 em.

@line-height-xl

Line height is 1.6 em.

@line-height-xxl

Line height is 1.8 em.

@line-height-xxxl

Line height is 2 em.
