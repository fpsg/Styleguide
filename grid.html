Grids

An introduction to the grid system we use at foodpanda.

Note: As we want to make our site friendlier for mobile users, and the current grid system hasn't been changed for a while, this page might change a lot in the near future.

Container size

Our main container is fluid and adjusts to the viewport up until a viewport size of 768 px or bigger. After that it's fixed and changes on set media-queries. Within the container there is a standard grid system with 12 columns.
